🧟 Monstruos de Halloween 🎃

Explora la API de Monstruos más terrorífica, donde encontrarás una gran variedad de criaturas con sus atributos, poderes y debilidades. ¡Autenticación segura y estadísticas incluidas!

Ver Funcionalidades

Funcionalidades de la API

Registro de Monstruos 👹

Agrega nuevos monstruos con sus respectivos poderes, debilidades y nivel de peligro.

Filtrado por Atributos 🧛

Busca monstruos filtrando por nombre, tipo, nivel de peligro, debilidades y poderes.

Estadísticas 📊

Consulta estadísticas detalladas sobre el nivel de peligro y tipo de monstruos en la base de datos.

Top Monstruos Más Terroríficos 👻

Descubre el top 10 de los monstruos más peligrosos y temibles.

Monstruo Aleatorio 🎲

Obtén un monstruo al azar y conoce sus habilidades especiales y debilidades.

Comparación de Monstruos ⚔️

Compara las características de dos monstruos para ver sus fortalezas y debilidades.

Búsqueda por Habilidad Especial 🧙‍♂️

Busca monstruos por habilidades especiales como invisibilidad o control mental.

Acerca de

Esta API ha sido desarrollada, pensada para la gestión y exploración de criaturas míticas de Halloween. El proyecto incluye autenticación mediante tokens y documentación interactiva con Swagger.
